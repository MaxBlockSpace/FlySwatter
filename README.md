## Quick Start

1. Clone the repository
2. Set up the environment:
```bash
chmod +x setup.sh
./setup.sh
```
3. Update `.env` with your API credentials
4. Activate the virtual environment:
```bash
source venv/bin/activate
```
5. Start the bot:
```bash
python src/main.py
```
